FTXX desktop

Lightweight "First to X" score tracker for streams and locals. Point OBS at it, change any settings you want, and off you go.

↓ Windows ↓ Linux
  • Set any "First to" goal — FT5, FT10, FT50, whatever
  • Score color shifts toward your accent as you approach the goal
  • Load any .ttf or .otf font to match your stream overlay
  • Settings persist between sessions
slpmeup web app

Browse Melee tournament replays, watch rendered videos, and download .slp files — all from one place.

slpmeup.xyz ↗
  • Search by tournament, player, or character
  • Replays automatically converted to video and uploaded to YouTube
  • Download original .slp files to watch in Slippi
  • Per-game stats like stocks, characters, and stage
reswiss web app

Run Swiss-format brackets with automatic pairings, tiebreakers, and no rematches.

swiss.slpmeup.xyz ↗
  • Automatic round-by-round pairings based on records
  • Buchholz tiebreakers for fair final standings
  • No rematches — players never face the same opponent twice
  • Best-of-3 or best-of-5 format
StreamFighter desktop

Desktop scoreboard manager for tournament streams. Update player names, scores, characters, and casters — changes push to OBS overlays in real time.

  • Pull brackets and sets directly from start.gg
  • Player and caster presets so you don't re-type regulars
  • Character and costume portraits on the overlay
  • Customizable overlay colors, fonts, and layout
  • Game pack system — add any game, not just Melee
