~/projects

Things I've built, broken, and occasionally shipped.

slpmeup Go

Pipeline for processing Melee .slp replays into YouTube uploads with a browsable replay viewer site.

  • Discord bot with SQLite job queue
  • Video pipeline via slp2mp4 + ffmpeg + Xvfb
  • Vanilla TS frontend compiled with esbuild
  • Caddy reverse proxy on Hetzner VPS
swiss-bracket React + Bun

Swiss bracket management tool for Melee tournaments with Burstein pairing and Buchholz tiebreakers.

  • Rematch prevention and bye handling
  • Alternating standings/pairings panels
  • Started as a Bubble Tea TUI, rebuilt for web
ftxx Go + Fyne
screenshot

Desktop "First to X" score tracker for 1v1 sets with live accent-color lerp, custom fonts, and persistent settings.

Linux Windows
  • Two-player panels with +1/-1/Reset controls
  • Score text color interpolates white → accent as you approach the goal
  • Custom .ttf/.otf font loading and configurable UI scale
  • Cross-compiled for Linux/Windows via fyne-cross
skillet Go CLI

Checksum tool supporting MD5/SHA1/SHA256/SHA512 via streaming io.MultiWriter. Because "making hash."

  • Charm ecosystem naming conventions
  • Planned RetroAchievements integration
  • Console-specific preprocessing subcommand
